# Taj Alwaqar

This is a mobile app built on Expo React Native.
The purpose of this app is to provide Quran memorization assistance by allowing users to create plans for audio playback.

## Features

### App 1: Mobile App "Taj Alwaqar"

#### Audio playback

1. Selection of sheikh recitation recordings

#### Memorization plans generation 

1. Pre-configured memorization styles / easy setup
1. Support resuming from where you stopped last time
1. Memorization style configuration:
    1. Add extra pauses in long verses / ayat
    1. Pause duration multiplier
    1. Setting of daily goal (Daily Wird ورد)
    1. Quick review before reaching goal

#### Resume support for memorization plans

### App 2: Audio Analyzer

#### Analyze long verses / ayat and generate pause pointers
