# Taj Alwaqar

This is a mobile app built on Expo React Native.
The purpose of this app is to provide Quran memorization assistance by creating plans for audio playback.



# Development toolkit notes

## Dev Tools on a Windows machine

### Mobile App

1. Node JS
2. Visual Studio Code
3. Expo: npm install -g expo-cli
4. Android Studio
5. Java JDK
6. Git
7. (Optional) AI tool (Antigravity, etc.)


### Audio Analyzer

1. .NET 8.0
2. NAudio
3. Visual Studio
